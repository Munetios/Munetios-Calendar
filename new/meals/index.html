Help

How to add a meal Open the add button, enter a title, type, ingredients, notes, and optionally attach an image file.
How bookmarks work Use the bookmark button on any meal card. Saved meals appear in the bookmarks modal and in bookmarked filtering.
How the calendar works Calendar shows meals by real month dates. Use previous and next controls to move between months.
Where is data stored? Meals and preferences are stored locally in IndexedDB and local storage.
Can I move data to another device? Yes. Use export in settings, then import the JSON file on another device.
What counts as past? Any meal created before today moves into past automatically.
